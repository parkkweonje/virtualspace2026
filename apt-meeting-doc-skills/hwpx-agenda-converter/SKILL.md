# 안건제안서 변환 스킬

## 목적
삼부4단지 아파트 대표회의 안건 HWPX 파일을 읽어 **안건제안서** 형식의 새 HWPX 파일로 변환한다.

## 도구
- `scripts/extract_hwpx.py` — HWPX에서 텍스트 추출
- `scripts/create_hwpx.py` — 안건제안서 HWPX 생성

## 안건제안서 고정 정보
- **단지명**: 삼부4단지 아파트
- **제안자**: 입주자대표회의
- **일자**: 변환 당일 (또는 원본 파일의 일자)

---

## 작업 순서

### 1단계: HWPX 파일 위치 확인
사용자가 제공한 HWPX 파일 경로를 확인한다. 여러 파일이면 하나씩 처리하거나, 사용자에게 어느 파일부터 할지 확인한다.

### 2단계: 텍스트 추출
```bash
python <스킬경로>/scripts/extract_hwpx.py "<입력파일.hwpx>" "<추출결과.json>"
```
추출 결과에서 `paragraphs` 배열을 읽어 내용을 파악한다.

추출이 실패하거나 텍스트가 없으면 → `Preview/PrvText.txt` 내용을 fallback으로 사용한다.

### 3단계: 내용 분석 및 안건제안서 구조로 재편성
추출된 텍스트를 분석하여 아래 JSON 구조로 정리한다:

```json
{
  "안건번호": "제X안",
  "제목": "안건 제목",
  "제안자": "입주자대표회의",
  "일자": "2026. 06. 11.",
  "제안이유": "왜 이 안건을 제안하는지 (배경, 필요성)",
  "현황및문제점": "현재 상황 및 문제점 분석",
  "소요예산": "금액 및 예산 내역 (없으면 '해당 없음')",
  "기대효과및향후계획": "안건 처리 후 기대 효과와 추진 일정"
}
```

**분석 지침:**
- 원본에 명시된 내용을 최대한 보존하되, 안건제안서 항목에 맞게 배치한다
- 원본에 예산 정보가 없으면 `소요예산`은 "해당 없음"으로 기재
- 원본에서 날짜를 찾을 수 없으면 오늘 날짜 사용
- 제목이 파일명에 있으면 그것을 우선 사용 (예: "5안 410동 입출구 울타리 보수" → "410동 입·출구 울타리 보수에 관한 건")
- 안건번호는 파일명에서 추출 (예: "5안" → "제5안")

### 4단계: 안건 데이터를 JSON 파일로 저장
```bash
# agenda_data.json 으로 저장 (작업 디렉토리에)
```

### 5단계: HWPX 생성
```bash
python <스킬경로>/scripts/create_hwpx.py "<agenda_data.json>" "<출력파일명_안건제안서.hwpx>"
```

출력 파일명 규칙: `{원본파일명}_안건제안서.hwpx`
예: `5안_410동울타리보수_안건제안서.hwpx`

### 6단계: 결과 확인 및 전달
생성된 HWPX 파일을 `present_files`로 사용자에게 전달한다.
여러 안건을 처리한 경우 모든 파일을 한 번에 전달한다.

---

## 여러 파일 일괄 처리

사용자가 여러 HWPX 안건 파일을 한꺼번에 변환 요청 시:
1. 각 파일을 순서대로 처리 (2~5단계 반복)
2. 모든 출력 파일을 한 번에 전달

---

## 주의사항

- HWPX는 ZIP 기반이므로 `zipfile` 모듈로 읽는다
- Section XML 추출에 실패하면 `Preview/PrvText.txt`를 대신 사용
- 생성된 HWPX는 한글(HWP) 프로그램에서 열 수 있다
- 원본 파일의 첨부 자료(표, 이미지)는 텍스트만 추출되므로 주요 수치는 원본 참고
