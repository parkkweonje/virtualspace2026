# 출력 서식 템플릿

아카이브에 실제 과거 문서가 있으면 **과거 문서의 구조·문체가 이 템플릿보다 우선**한다.
아카이브가 비어 있을 때 아래 기본 서식을 사용한다.

---

## 1. 안건자료 (안건제안서) 서식

```
삼부4단지 아파트 입주자대표회의

              안  건  자  료

안 건 번 호 : 제[ ]안
제        목 : ○○○○에 관한 건
제  안  자 : 입주자대표회의 (또는 관리사무소)
제 안 일 자 : 2026. 07. 02.

──────────────────────────────

1. 제안 이유
   □ [배경 및 필요성 — 왜 지금 이 안건이 필요한가]

2. 현황 및 문제점
   □ 현황
     ○ [현재 상태, 관련 수치]
   □ 문제점
     ○ [방치 시 발생하는 문제]

3. 세부 내용
   □ [추진 방식, 범위, 일정]
   □ 사업자 선정 방식: [경쟁입찰 / 수의계약] (「주택관리업자 및 사업자 선정지침」 근거)

4. 소요 예산
   □ 예산액: [        ]원 (부가세 포함 여부 명시)
   □ 재원: [장기수선충당금 / 관리비 / 잡수입]
   ※ 장기수선충당금 사용 시 장기수선계획 반영 여부 확인 필수

5. 근거 규정
   □ 공동주택관리법 제○조, 같은 법 시행령 제○조
   □ 관리규약 제○조

6. 기대 효과 및 향후 계획
   □ [기대 효과]
   □ [의결 후 추진 일정]

──────────────────────────────
                                                끝.

                    2026. 07. 02.
                    삼부4단지 아파트 입주자대표회의
```

### 카테고리별 주의점
- **공사**: 장기수선계획 대상 여부 → 대상이면 수시조정 필요성 검토 문구 포함
- **용역·사업자 선정**: 선정지침상 입찰 방법·낙찰 방법 명시
- **예산**: 재원 구분(장기수선충당금/관리비/잡수입) 반드시 구분
- **규정 개정**: 신구조문 대비표 형태 권장

## 2. 회의록 초안 서식

```
제[ ]회 [정기/임시] 입주자대표회의 회의록

1. 회의 개요
  가. 일    시 : 2026. ○. ○.(○) ○○:○○
  나. 장    소 : 관리사무소 회의실
  다. 재적인원 : [ ]명 / 참석 [ ]명 / 불참 [ ]명 → 성원 [충족/미충족]

2. 참석자
  □ 입주자대표회의: 회장, 감사, ○동 대표 ... [직책명만 기재]
  □ 배석: 관리사무소장 외 [ ]명

3. 개회
  □ 회장이 성원 보고 후 개회를 선언함 (○○:○○)

4. 안건 심의

  【제1안】 ○○○○에 관한 건
   □ 제안 설명: [관리사무소장/제안자]가 안건자료에 따라 설명함
     ○ [핵심 내용 요약]
   □ 주요 질의 및 논의
     ○ [예상 논의 사항 — 과거 유사 안건 논의 패턴 기반, 초안임을 표시]
   □ 의결 결과: [원안 가결 / 수정 가결 / 부결 / 보류]
     ○ 찬성 [ ]명, 반대 [ ]명, 기권 [ ]명

  【제2안】 ... (안건 수만큼 반복)

5. 기타 논의사항
   □ [        ]

6. 폐회
  □ 회장이 폐회를 선언함 (○○:○○)

                    2026. ○. ○.

   회    장  ○○○  (서명)
   감    사  ○○○  (서명)
   회의록 작성자  ○○○  (서명)
```

### 회의록 초안 작성 규칙
- 의결 결과·표결 수는 **절대 임의 기재 금지** — 빈칸 `[ ]` 유지
- "주요 질의 및 논의"는 과거 유사 안건에서 반복된 쟁점을 예상 논의로 제시하되, 문서 상단에 "※ 본 문서는 초안이며, 실제 회의 내용에 따라 수정 필요" 문구 명시
- 개인 성명은 서명란 외에는 직책명으로 대체
