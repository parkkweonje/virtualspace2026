# 입주자대표회의 문서 생성 스킬 모음

삼부4단지 아파트 입주자대표회의 문서 작업을 자동화하기 위해 Claude Code 세션에서 만든
스킬(코드) 백업입니다. 원래 세션의 임시 환경(`~/.claude/skills/`)에만 있어 저장소에
남지 않던 것을 여기 보관합니다.

## 포함 스킬

### 1. `apt-meeting-docs`
3년치 과거 자료(안건·안건자료·회의록) 아카이브를 근거로 문서를 자동 생성한다.

- **모드 A — 아카이브 구축·갱신**: hwpx/docx/pdf 과거 자료를 표준 마크다운으로 분류·정리하고 `index.json` 생성
- **모드 B — 안건 → 안건자료(안건제안서) 생성**: 유사 과거 안건을 검색해 문체·구조·예산 규모를 반영
- **모드 C — 공고문 → 회의록 초안 생성**: 과거 회의록 패턴을 학습해 초안 작성(의결 결과는 빈칸 처리)

구성:
- `scripts/extract_hwpx.py` — HWPX 텍스트 추출
- `scripts/build_index.py` — 아카이브 인덱스(`index.json`) 생성
- `references/formats.md` — 안건자료·회의록 출력 서식 템플릿
- `references/archive/` — 안건/회의록 아카이브(현재 비어 있음, `index.json` 항목 0건)

### 2. `hwpx-agenda-converter`
대표회의 안건 HWPX 파일을 공식 **안건제안서** 형식 HWPX로 변환한다.
(제안이유 / 현황 및 문제점 / 소요예산 / 기대효과 및 향후계획 구조로 재편성)

구성:
- `scripts/extract_hwpx.py` — HWPX 텍스트 추출
- `scripts/create_hwpx.py` — 안건제안서 HWPX 생성

## 사용법
각 스킬 폴더의 `SKILL.md`에 모드 판별·작업 순서·출력 서식이 상세히 기술되어 있다.
스킬로 다시 설치하려면 해당 폴더를 `~/.claude/skills/` 아래로 복사하면 된다.

## 참고
- 다루는 문서 유형: 입주자대표회의 **공고문 · 안건제안서 · 회의록** (자동 생성)
- **입찰공고문**은 별도 코드 스킬이 아니라 문서 작업물로 관리되어 이 백업에는 포함되지 않는다.
